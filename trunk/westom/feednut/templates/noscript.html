WARNING : JAVASCRIPT IS NOT ENABLED IN YOUR BROWSER!

Javascript is required to use many of the features of this site. The following instructions will help you enable Javscript in your browser. Simply select your browser from the list below and follow the instructions. When complete you can return to Feednut.com.

Please select your browser:
  • Firefox
  • Internet Explorer
  • Netscape
  • Safari
  • Opera

Firefox

Please follow the instructions appropriate to the version of your browser:

1.X

  1. Select theTools menu and then select Options.
  2. Select Web features.
  3. Check the box beside Enable Javascript.
  4. Click OK to close the Options window.
  5. Return to Feednut.com.

Internet Explorer

Please follow the instructions appropriate to the version of your browser:

6.X

  1. Select the Tools menu and then select Internet Options.
  2. Select the Security tab.
  3. Click the Custom level button. The Security Settings window will appear.
  4. In the Scripting category do the following:
    • enable Active scripting
    • enable Allow paste options via script
    • enable Scripting of Java applets
  5. Click OK to close the Security settings window.
  6. Click OK to close the Internet Options window.
  7. Return to Feednut.com.

5.X

  1. Select the Tools menu and then select Internet Options.
  2. Select the Security tab.
  3. Click the Custom level button. The Security Settings window will appear.
  4. In the Scripting category do the following:
    • enable Active scripting
    • enable Allow paste options via script
    • enable Scripting of Java applets
  5. Click OK to close the Security settings window.
  6. Click OK to close the Internet Options window.
  7. Return to Feednut.com.

5.X for Mac OS X

  1. Select the Explorer menu and then select Preferences.
  2. Click the arrow next to Web Browser.
  3. Click Web Content.
  4. In the Active Content check Enable scripting.
  5. Click OK to close the Preferences window.
  6. Return to Feednut.com.

5 for Mac OS 9

  1. Select the Edit menu and then select Preferences.
  2. Click the arrow next to Web Browser.
  3. Click Web Content.
  4. In the Active Content check Enable scripting.
  5. Click OK to close the Preferences window.
  6. Return to Feednut.com.

Netscape

Please follow the instructions appropriate to the version of your browser:

7.X

  1. Select the Edit menu and then select Preferences.
  2. Click the arrow next to Advanced.
  3. Click Scripts & Plugins.
  4. Check Navigator beneath "Enable Javascript for".
  5. Click OK.
  6. Return to Feednut.com.

6.X for Mac OS X

  1. Select the Edit menu and then select Preferences.
  2. Click Advanced.
  3. Check Enable Java.
  4. Check Enable JavaScript for Navigator.
  5. Click OK.
  6. Return to Feednut.com.

Safari

Please follow the instructions appropriate to the version of your browser:

2.X

  1. Select the Safari menu and then select Preferences.
  2. Click the Security .
  3. Check Enable JavaScript.
  4. Close the window.
  5. Return to Feednut.com.

Opera

Please follow the instructions appropriate to the version of your browser:

8.X

  1. Select the Tool menu and then select Preferences.
  2. Click the Advanced tab.
  3. Check Enable JavaScript.
  4. Click OK.
  5. Return to Feednut.com.

6.X / 7.X

  1. Select the File menu and then select Preferences.
  2. Select Multimedia.
  3. Check Enable JavaScript.
  4. Click OK.
  5. Return to Feednut.com.
